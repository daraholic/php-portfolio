# 個人作品集管理系統

---
## 功能要求
**前台：**
1. nav bar點擊跳轉
2. 個人介紹
3. 作品集區塊
4. 作品能以連結跳轉
5. 頁尾區塊
6. RWD功能

**後台：**
1. 登入功能
2. 後台有以下功能
   * 個人基本資料管理
     * 個人照片
     * 自我簡單介紹
   * 作品管理
     * 可新增作品項目
     * 可分類作品
     * 可上傳作品縮圖
     * 作品列表
     * 作品項目的更新/刪除
   * 網站管理
     * 可設定網站部份文字或內容變更
     * 可透過設定改變網站部份視覺
       * 頁首 / 頁尾 / 背景圖版
     * 可設定網站各區塊或項目的顯示與否
